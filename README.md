# bpm10-waldbinnenklima
R-Auswertungspipeline zur Analyse langfristiger Änderungen des Waldbinnenklimas in unterschiedlich bewirtschafteten Wäldern – auf Basis der Klima- und ForMI-Daten der Biodiversitäts-Exploratorien (BExIS). HAWK-Projektarbeit BPM 10.
Hinweis: Die R-Skripte dieser Auswertungspipeline wurden mit Unterstützung von KI (Claude Sonnet 4.8, Anthropic) erstellt und anschließend fachlich geprüft und angepasst.
